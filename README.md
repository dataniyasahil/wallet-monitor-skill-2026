# Wallet Monitor Skill

A custom IronClaw skill for monitoring NEAR wallet balances and setting price alerts.

## Features

- Check wallet balances across multiple tokens
- Set price alerts with conditions
- Generate daily summary reports
- Send notifications via Telegram

## Installation

skill_install name='wallet-monitor'

## Usage

check-balance
set-alert NEAR above 4.00
list-alerts

- SKILL.md - Skill definition
